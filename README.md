## Installation
Get Benbo Quest from https://thumbtack.itch.io/benboquest and toss the `data.win` file in `ports/benboquest/gamedata` as `game.droid`.

## Default Gameplay Controls
| Button | Action |
|--|--|
|START|Confirm|
|D-PAD|Move|
|LEFT ANALOG|Move|
|A, B|Jump|
|X, Y|Shoot|
